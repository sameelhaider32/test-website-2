# Local Image Replacement Guide

This storefront now expects local images in:

- `public/images/products/`
- `public/images/collections/`
- `public/images/hero/`
- `public/images/placeholders/`

If you replace files, keep the same filenames to avoid code changes.

## Notes
- Use bedding-appropriate imagery per product type.
- You can replace `.svg` files with `.jpg` or `.webp`, but then update the data path references.
- Keep dimensions reasonably high (recommended min width: 1400px for hero/collection, 1200px for products).
